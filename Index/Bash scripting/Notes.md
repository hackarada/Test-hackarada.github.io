# Notes

---

bash script should always start with  (shebang ) 

    #!/bin/bash 

Hello world 

    #!/bin/bash
    echo hello world

Variables

    #!/bin/bash
    #A variable is declared without a $, but has a $ when invoked.
    #there must not be a space between variable and value.
    mesg="hello world"
    echo $mesg
